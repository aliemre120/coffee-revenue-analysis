# coffee-revenue-analysis
Business Analytics project analyzing revenue growth dynamics in a multi-location coffee shop chain.

Project Overview

This project analyzes transaction-level sales data from a multi-location coffee shop chain to evaluate revenue growth dynamics, category concentration risk, and demand stabilization signals.

Key Insights

Revenue increased 104% from January to June.

Growth is volume-driven, not price-driven.

Coffee and Tea account for 66.7% of total revenue.
